# BBABAM 문서 센터

이 폴더는 `빠밤` 프로젝트의 개발/디자인 협업을 위한 기준 문서입니다.

## 문서 목록

- `docs/pencil-brief.md`: Pencil에게 바로 전달할 작업 브리프
- `docs/architecture.md`: 현재 프로젝트 구조와 기술 스택
- `docs/games-design-guide.md`: 미니게임 디자인 방향성과 디테일 가이드
- `docs/handoff-checklist.md`: 작업 완료 전 점검 체크리스트

## 빠른 시작

1. 디자인/UX 작업을 시작할 때는 `docs/pencil-brief.md`부터 확인합니다.
2. 코드 구조를 파악할 때는 `docs/architecture.md`를 확인합니다.
3. 게임별 톤앤매너는 `docs/games-design-guide.md`를 기준으로 맞춥니다.
4. PR 전에는 `docs/handoff-checklist.md`로 최종 점검합니다.
