# Pencil 작업 브리프

## 프로젝트 개요

- 프로젝트명: `빠밤 (BBABAM)`
- 서비스 성격: 아기 성별 공개 카드/미니게임 기반의 감성형 모바일 웹 경험
- 핵심 목표: 사용자가 "귀엽고 따뜻하다", "축하 분위기가 난다"를 즉시 느끼는 비주얼 완성도

## 가장 중요한 디자인 요구사항

아래 2가지는 모든 화면/컴포넌트에서 반드시 지켜주세요.

1. **한국인 사용자가 좋아하는 귀여운 감성**
   - 과하게 유아틱하지 않고, "사랑스럽고 세련된" 귀여움
   - 둥근 형태, 부드러운 그림자, 파스텔 중심의 안정적 색조
   - 너무 강한 원색 대비/자극적인 네온톤은 지양

2. **아동제품(베이비/키즈) 느낌**
   - 축하/선물/파티 무드가 보이되 과도한 장난감 느낌은 피함
   - 안전하고 포근한 인상(soft, warm, clean)
   - 아이콘/일러스트는 "말랑한 질감"과 "친근한 표정/모양" 선호

## 타겟 톤앤매너

- 감성 키워드: `귀여움`, `포근함`, `축하`, `신뢰감`, `깨끗함`
- 피해야 할 키워드: `공포`, `어두움`, `고딕`, `딱딱함`, `차가운 금속감`
- 문구 톤: 짧고 다정한 한국어, 과한 밈/슬랭 사용 금지

## 디자인 시스템 가이드 (초안)

- Radius: `xl~3xl` 위주
- Shadow: 연하고 넓은 그림자(하드 섀도우 금지)
- Color:
  - Primary Pink: `#F472B6` 계열
  - Sky Blue: `#60A5FA` 계열
  - Cream: `#FFF7ED` 계열
  - Mint Accent: `#6EE7B7` 계열
- Typography:
  - 제목: 둥글고 친근한 볼드
  - 본문: 가독성 높은 산세리프

## 게임별 우선순위

현재 대상 게임:

- `ice-cream`
- `balloon`
- `scratch`
- `roulette`
- `gift-box`
- `gacha`

우선순위:

1. 첫인상 비주얼(아이콘성)
2. 탭/스크래치/회전 인터랙션의 쾌감
3. 완료 순간의 축하 연출(짧고 강하게)

## 구현 시 요청사항

- 기존 게임 로직은 최대한 유지하고, 비주얼/모션 중심으로 개선
- 모바일 우선(`9:16` 감성), 작은 화면에서도 레이아웃 안정성 확보
- 한 요소를 과하게 복잡하게 만들기보다, 전체 톤 통일을 우선

## 완료 정의 (Definition of Done)

- 6개 게임의 시각 언어가 하나의 브랜드처럼 보일 것
- 사용자 피드백 기준 "귀엽다/완성도 높다" 반응 확보
- 린트 에러 없음, 기본 동작(탭/클릭/완료 콜백) 정상 동작
