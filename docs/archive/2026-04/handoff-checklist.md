# 핸드오프 체크리스트

## 기능 안정성

- [ ] 게임 완료 시 `onComplete`가 정상 호출된다.
- [ ] 탭/스크래치/드래그 인터랙션이 모바일에서 정상 동작한다.
- [ ] 애니메이션 중 상태 꼬임(중복 클릭, 멈춤)이 없다.

## 디자인 품질

- [ ] 한국형 귀여운 감성(포근/말랑/정돈)과 어울린다.
- [ ] 아동제품 느낌(안전/따뜻함/밝음)이 난다.
- [ ] 6개 게임의 시각 언어가 통일되어 보인다.

## 접근성/가독성

- [ ] 텍스트 대비가 충분하다.
- [ ] 중요한 인터랙션 요소가 직관적으로 보인다.
- [ ] 작은 화면에서도 레이아웃 깨짐이 없다.

## 개발 품질

- [ ] `npm run lint` 통과
- [ ] `npm run build` 통과
- [ ] 변경사항이 문서(`docs`)와 동기화되어 있다.
