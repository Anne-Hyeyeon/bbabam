# 게임 디자인 가이드

## 공통 비주얼 원칙

- 둥근 실루엣 + 파스텔 컬러 + 부드러운 명암
- 요소 간 간격을 넉넉하게 유지해 답답함 방지
- 한 화면 내 포인트 컬러는 2~3개로 제한
- 완료 이펙트는 화려하되 1초 내외로 짧게

## 게임별 가이드

### 1) 아이스크림 퍼먹기 (`ice-cream`)

- 바닐라/크림 톤 중심의 깨끗한 스쿱 형태
- 숟가락 이동 궤적이 자연스럽게 보이도록 곡선 모션 사용
- 퍼먹힌 자국은 원형보다 타원형 + 미세한 그림자로 구현

### 2) 풍선 터뜨리기 (`balloon`)

- 검은 풍선 바디 + 블루/핑크 레터링으로 젠더리빌 무드 강화
- 풍선 광택은 2단 하이라이트로 부드럽게
- 진행에 따라 크기 증가 시 과도한 왜곡 금지

### 3) 스크래치 복권 (`scratch`)

- 스크래치 레이어는 실버 포일 질감 유지
- 내부 보상 슬롯은 명확한 대비(읽기 쉬운 아이콘)
- 스크래치 반경은 너무 작지 않게 설정해 스트레스 최소화

### 4) 룰렛 (`roulette`)

- 원판 외곽 링(골드/크림 톤)으로 고급감 부여
- 포인터는 상단 중앙 고정, 시인성 최우선
- 회전 종료 시 "멈춘 지점"이 직관적으로 보이게 구성

### 5) 선물상자 열기 (`gift-box`)

- 본체/뚜껑/리본 레이어 구분이 명확해야 함
- 리본 제거 -> 뚜껑 오픈 -> 축하 연출 순서가 즉시 이해되어야 함
- 축하 이펙트는 별/컨페티 계열로 통일

### 6) 뽑기 (`gacha`)

- 머신 창, 레버, 배출구의 구조를 명확히 분리
- 레버 당기는 동작의 반응성을 높여 만족감 강화
- 캡슐 등장/오픈 전환 타이밍을 명확히 구분

## 문구 가이드 (한국어)

- 짧고 다정한 안내 문구 선호
- 예: "한 번 더 톡!", "거의 다 왔어요!", "짜잔!"
- 명령형 문구도 부드럽게 표현

## QA 체크포인트

- 모바일에서 터치 타겟이 충분히 큰가
- 색약 환경에서도 핵심 인터랙션이 구분되는가
- 한 게임만 튀지 않고 전체 톤이 통일되는가
